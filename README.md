# 快速开始

``` javascript
npm i
npm run test
```

- 通过`npm run test`来快速生成快照，同时`imageSnapshot`会帮助我们通过编写的story生成对应的快照图

- 对代码进行一次更改,例如：`src/stories/Page/tsx`，`server`会重新启动，然后在stories目录下输出diff
